<script>

alert(`1 + 3 + 2 is ${1+3+2}`);
</script>

자료형

  • 숫자형 - 정수, 부동 소수점 숫자다. (±253)
  • bigint - 제약이 없는 정수, 끝에 b를 붙인다.
  • 불린형 - true, false, 논리연산자로도 표현 가능하다.
  • null - 다른 언어와는 별개로 알수 없는 값이라는 뜻으로, 독립적인 자료형이다.
  • undefined - 아직 할당 되지 않은 값, 다른 언어의 쓰레기값대신 나오는 자료형
  • 객체형 - 복잡한 데이터 구조를 표현할 때 사용, null도 typeof연산자를 쓰면 object로 나오긴 하나, 객체는 아님.
  • 심볼형 - 객체의 고유 식별자를 만들 때 사용한다.

(※ typeof(x) 혹은 괄호가 아닌 띄어쓰기로 x를 넣어도 된다.)

입력받기

let userInput = prompt('입력한 대로 화면에 출력됩니다.', '')

(※ 여기서 취소를 누를 경우에 null이 나오는 이유가 무엇인가?)

prompt의 리턴으로 '아무것도 입력하지 않음'을 받았기 때문에 null이 되는 것이다.
